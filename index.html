Connect Tiara to your GitHub 🎀

Your board lives safely in your own GitHub account. Connect it once and you never download anything again — just visit this page.

  1. Open github.com/settings/tokens → Generate new token (fine-grained).
  2. Under Repository access pick All repositories (or just your app repos + a new tiara one).
  3. Under Permissions → Repository, set Contents to Read and write (and, if you'd like it to make the board repo for you, Administration → Read and write too). Generate, then copy the token.
  4. Paste it below and press Connect.
