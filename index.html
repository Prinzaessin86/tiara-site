Connect Tiara to your GitHub

Your board lives safely in your own GitHub account. Connect it once and you never download anything again — just visit this page.

  1. Open github.com/settings/tokens → Generate new token (classic).
  2. Name it "Tiara", then tick just two boxes: repo and project.
    repo = read/write your issues + board data · project = drive the GitHub boards
  3. Click Generate token and copy it (starts with ghp_…).
  4. Paste it below and press Connect.
